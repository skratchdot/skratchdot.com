### Build Info ###

1. Clone the repo

        git clone https://github.com/skratchdot/skratchdot.github.com.git

2. Checkout the "jekyll" branch

        git checkout jekyll

3. Update modules

        npm install

4. Build site by running grunt:

        grunt
